Automatically solves a game of wordle by itself.
1. Inputs the word "earth"
2. Start a temporary word. For each of the resulting characters, input each letter of the alphabet, 
and if the result character is a '#', keep the same letter in the same position.
3. Input temporary word into next guess and keep repeating until game is over.

USAGE: type ./wordle_ecs36b_s2023 into the terminal and select the game you want.

Assumption: no limit for amount of guesses.
